# 30/60/90 Implementation Roadmap

Verification date: 2026-03-05

## Summary
Roadmap is reliability-first and additive-only. Each increment ends with green validation gates and compatibility checks.

## 0-30 Days (Increment A)
### Deliverables
1. Cost Guardrails (CLI + summary keys + policy decisions).
2. Auto Job-Log Sync (schema migration + prefilled save flow).

### Required validation gate
- `dart run tooling/validate_agent_docs.dart`
- `dart run tooling/validate_workspace_hygiene.dart`
- `python -m compileall src tests`
- `python -m pytest -q tests/test_run_report.py tests/test_cli_flags.py tests/test_openai_transport_retries.py tests/test_retry_reason_mapping.py`

### Exit criteria
1. Cost estimate status always explicit.
2. Budget policy outcomes logged in `run_summary.json`.
3. Job-log save path persists new metrics when available.

## 31-60 Days (Increment B)
### Deliverables
1. Quality Risk Scoring engine.
2. Review Queue panel + CLI review export.

### Required validation gate
- Prior gate +
- `python -m pytest -q tests/test_translation_diagnostics.py tests/test_workflow_logging_safety.py tests/test_workflow_parallel.py`
- New tests for queue thresholding and deterministic scoring.

### Exit criteria
1. Deterministic scores across repeated runs.
2. Review queue exports reproducibly.
3. No regression in translation output assembly.

## 61-90 Days (Increment C)
### Deliverables
1. Queue/Batch Runner with resume and failed-only rerun.
2. Guided OCR/Image Advisor integrated into analyze/pre-run UX.

### Required validation gate
- Prior gate +
- `python -m pytest -q tests/test_checkpoint_resume.py tests/test_workflow_ocr_routing.py tests/test_workflow_ocr_availability.py`
- New queue runner and advisor test suites.

### Exit criteria
1. Queue resume is crash-safe and idempotent.
2. Failed-only rerun targets only failed units.
3. Advisor recommendations are explainable and non-forcing.

## Cross-Increment Guardrails
1. No public signature break in `TranslationWorkflow` entrypoints.
2. Backward compatibility retained for existing run artifacts.
3. Any contract-changing fix isolated in dedicated commit with explicit PR note.
